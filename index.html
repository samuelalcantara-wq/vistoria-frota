📊 Sincronização com Google Sheets

Cole abaixo a URL do Web App do Google Apps Script. Depois de configurado, todas as vistorias serão salvas na sua planilha automaticamente e sincronizadas entre celular e computador.

Você obterá essa URL ao publicar o Apps Script como Web App.
📋 Como configurar (uma vez só): 1. Abra o arquivo AppsScript_Vistoria.gs que foi entregue junto com este portal
2. Acesse script.google.com → Novo projeto → cole o código
3. Clique em Implantar → Nova implantação → Web App
4. Em "Executar como": Eu · "Quem pode acessar": Qualquer pessoa
5. Clique em Implantar e copie a URL gerada aqui
